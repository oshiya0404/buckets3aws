- Only PDF files should be uploaded.
- Content-Type should be application/pdf.
- Maximum file size should be 10 MB.
- File keys should follow this pattern:
  lecture-notes/<user-id>/<timestamp>-<filename>.pdf
- The bucket is private by default.
- Block Public Access is enabled.
- Versioning is enabled.
- Default encryption is enabled.
- Downloads use pre-signed URLs that expire after 5 minutes.
